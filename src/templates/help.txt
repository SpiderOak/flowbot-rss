{{ ':small_blue_diamond:' | emoji }} @{{ botname }} watch <feedUrl>
will start watching a new RSS feed in this channel

{{ ':small_blue_diamond:' | emoji }} @{{ botname }} list
will list all feeds currently watched in this channel and assign each an id, which you can use to unwatch (below)

{{ ':small_blue_diamond:' | emoji }} @{{ botname }} unwatch <id>
will stop watching the feed, where <id> is the id shown in the list (see above)
